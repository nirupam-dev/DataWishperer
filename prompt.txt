Create a Python Streamlit application called DataWhisperer that acts as an advanced AI data analysis assistant. The application should allow users to upload CSV files, view data summaries, and ask natural language questions about their dataset. 

It must include a robust end-to-end analytical pipeline featuring:
1. LLM Integration: A backend that uses LLM inference to convert user questions into executable Python code (using libraries like pandas and matplotlib). Implement Groq as the primary LLM provider for speed, and integrate Ollama as a local fallback option.
2. Sandboxed Execution: A secure, sandboxed execution environment to safely run the Python code generated by the LLM.
3. Visualization: The ability to generate, capture, and render chart visualizations and data insights directly within the app based on the analytical results.
4. UI/UX: A high-fidelity, professional, and dynamic user interface with a cinematic aerospace-themed design. The UI should have a landing page and an interactive analytics workspace.
5. Error Handling: Built-in error handling and self-correction mechanisms if the generated code fails to execute properly.
